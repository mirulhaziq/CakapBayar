# PWA Icons Setup

To complete the PWA setup, you need to create two icon files:

1. **`public/icon-192x192.png`** - 192x192 pixels
2. **`public/icon-512x512.png`** - 512x512 pixels

## Quick Setup Options:

### Option 1: Use an Icon Generator
1. Visit https://realfavicongenerator.net/ or https://www.pwabuilder.com/imageGenerator
2. Upload a square logo/image (at least 512x512)
3. Generate the icons
4. Download and place them in the `public/` folder

### Option 2: Create Simple Icons
You can create simple colored square icons with the CakapBayar branding:
- Use orange (#f97316) or pink (#ec4899) background
- Add white text "CB" or the üçú emoji
- Export as PNG at the required sizes

### Option 3: Use a Design Tool
- Use Figma, Canva, or any design tool
- Create 512x512 and 192x192 versions
- Export as PNG
- Place in `public/` folder

The icons should be:
- Square (1:1 aspect ratio)
- PNG format
- Named exactly: `icon-192x192.png` and `icon-512x512.png`
- Placed in the `public/` directory

Once added, the PWA will be fully functional with "Add to Home Screen" capability!
